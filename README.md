# CyFinex — GitHub Pages Setup

## Your two files
- `index.html` — Landing page
- `cyfinex-demo.html` — Interactive dashboard demo

## Deploy in 5 steps

### 1. Create a GitHub repository
- Go to github.com → New repository
- Name it: `cyfinex` (your URL will be: yourusername.github.io/cyfinex)
- Set to **Public**
- Click **Create repository**

### 2. Upload your files
- Click **Add file → Upload files**
- Drag both `index.html` and `cyfinex-demo.html` into the window
- Click **Commit changes**

### 3. Enable GitHub Pages
- Go to **Settings** (top menu of your repo)
- Scroll to **Pages** in the left sidebar
- Under **Source**, select **Deploy from a branch**
- Branch: **main** / Folder: **/ (root)**
- Click **Save**

### 4. Wait 60 seconds, then visit:
https://yourusername.github.io/cyfinex

### 5. Optional — custom domain (cyfinex.io)
- Buy domain at namecheap.com (~$15/yr)
- In GitHub Pages settings, add your custom domain
- Follow GitHub's DNS instructions (~10 mins)
